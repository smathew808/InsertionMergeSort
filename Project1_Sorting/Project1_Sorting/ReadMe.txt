How to execute the program:

after starting the program, the first menu that will pop up will be a choice between the 3 sorting algorithims. the options will be between 1-3 on the keyboard.  

after a choice has been, another menu will pop up asking you to choose which file size you would like. The choices will be between 1-5. 

after that choice has been made, one more menu will pop up asking you to choose between a permuted list or a sorted list. The choices will be between 1-2.

Once all the choices have been made, the program will sort the lists and ouput the times on the console for you. There will also be an output txt file of the sorted list in the solution folder. It will also tell you that in the console as well.
